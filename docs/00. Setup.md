### Clone source code
```
git clone ...
cd /myproject
composer install
```

### Config virtual host
````
<VirtualHost *:80>
    DocumentRoot "D:\projects\research\symfony-learn\public"
    ServerName local.symfony.com

    <Directory "D:\projects\research\symfony-learn\public">
        Options +Indexes +FollowSymLinks -MultiViews
        AllowOverride None
        Order allow,deny
        Allow from all
        <IfModule mod_rewrite.c>
            RewriteEngine On
            RewriteRule ^/?([0-9]+)/([0-9]+)/?$ /index.html#$1/$2 [R=301,L,NE]
            RewriteCond %{REQUEST_FILENAME} !-f
            RewriteRule ^(.*)$ /index.php [L]
        </IfModule>
    </Directory>
</VirtualHost>
````

### Clear cache
